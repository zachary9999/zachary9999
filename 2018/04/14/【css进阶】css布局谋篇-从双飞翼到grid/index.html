不谋全局者，不足谋一域。对于css零碎的知识掌握了很多，那么如何做出一个漂亮的网页、网站。宏观上这就需要先来进行页面的布局。本文不做深入基础性知识讲解，只讲核心，方便速记和选择区分。

现在来实现一个经典的三列布局：高度固定，左右两列宽度固定，中间自适应。
左|中|右。
从双飞翼布局讲起，到最近的gird布局。
默认容器300px高度，左右两个模块宽都为150px。

一、圣杯布局

一句话概括就是:
1.容器设置左右padding分别为左右子模块的宽度,左右子元素向左浮动;2.左子元素margin：-100%+right:容器左padding距离；3.右侧子元素maring:-(容器右padding值)即可；中间保持不变。
