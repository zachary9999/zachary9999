css进阶之css水平垂直居中方案汇总

垂直居中是css之中绕不过去的坎之一（或者升级必须打的boss之一)。
所有效果示例汇集到一个html文件中。
https://zachary9999.github.io/css-advance-Align-VerticalCenter/

容器我们用section表示默认宽100%、高300px，居中的元素我们用.box样式,默认宽100px、高100px,代码展示最核心的。以下如不强调则为宽高自适应（不限宽高）。
默认布局如下

1.absolute、负边距 宽高指定

负边距在很多排版布局中常用需要多加练习理解。

2.absolute、calc函数 宽高指定

calc函数用于动态计算长度值。
需要注意的是，运算符前后都需要保留一个空格，例如：width: calc(100% - 10px)；
兼容性:IE9+

3.当absolute拉伸遇上margin:auto

absolute情况下left、right,top、bottom是可以替代width和height的，表现出拉伸特性,这种特性可以用来实现百分比高度(比如实现移动端的n*n的布局在此不深入展开)，但在遇到margin时就表现出居中特性。

4.inline-block

section容器line-height设置的是内联元素垂直居中，那么.box设置为display: inline-block;

5.absolute、transform

transform 属性向元素应用 2D 或 3D 转换。该属性允许我们对元素进行旋转、缩放、移动或倾斜。

6.writing-mode

IE5时代就有了的属性，不常用。如果要深挖请看张鑫旭的这篇文章：
改变CSS世界纵横规则的writing-mode属性

7.css table方案

table布局方案省略

8.flex方案

flex的重要性不言而喻，移动端常用布局选择。深入了解flex请看阮一峰的flex文章Flex布局教程：语法篇。

9.grid

与flex有异曲同工之妙的是grid属性表现性一至，align-，指垂直方向，justify-水平方向
